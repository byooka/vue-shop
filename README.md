### 技术栈
vue + vue-router + vue-table-with-tree-grid + vue-quill-editor + echarts + nprogress + element-ui


### 项目运行

```
git clone https://github.com/lumengxin/vue-shop.git
npm install
npm server                  // 运行
npm run lint                // 格式化代码
npm run build               // 打包上线
```
